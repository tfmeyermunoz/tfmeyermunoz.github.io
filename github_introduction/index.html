Hello everyone. First of all, thanks a lot to the Jonathan Mcglone's blog " Creating and Hosting a Personal Site on GitHub" (http://jmcglone.com/guides/github-pages/#).
I found there everything I needed to construct this guide and I hope it will help you to be able to create your own Github profile. Let's do it:


1. Create a Github profile.

- Go to the Github homepage (www.github.com) and tap the "sign up" button on the right corner of the screen. You will have to give your email account and create a username and a password.



2. Sign in with your new profile.

- Once you have created your profile, you have to sign in with your email account and password.


3. Create your personal repository.

- After signing in, tap on the "repositories" button in your profile. Once there, you will be able to tap on the "new" button and upload a new screen, where you will have to choose your file name



- Select as name "your_profile_name.github.io" as the repository name. For example, as my profile is tfmeyermunoz, my repository name is "tfmeyermunoz.github.io".
After that, choose if you want to have a README file to describe the repository and a gitignore file, to be able to ignore some of the files in your repository.

4. Upload the files youn want to show.


- Once you have created your personal repository, you will have to create your homepage. If you want to upload any picture or file to be able to use it as part of your homepage, you will have to do the following:


  • 1. Tap the "Add file" button.
  • 2. Select the "Upload files" alternative.
  • 3. Navigate through your computer and upload the files you need in your repository. For example, a profile picture.
  • 4. Commit your changes.


5. Create your index.html file


- In your repositories homepage, tap the "Add file" button and then click on "Create new file". After that, follow these instructions:


  • 1. Select "index.html" as file name.
  • 2. Create your profile as you wish. It is highly recommended to create a title and include a picture.
  • 3. Commit the changes.


- For example, in the picture above, I have already created a standard layour for my personal page, to be able to use it in multiple cases without repeating work.

6. See your results.


- Once you have commited the changes, you can go to your_profile_name.github.io and see your results.



- As shown in the picture, you will be able to create an interactive profile and try to make it as attractive as possible
